# GTSC Framework Crypto

This package contains helper methods and classes which implement cryptographic functions.

- Blake2B
- Sha256
- HmacSha1
- Ed25519
- Bip32
- Slip100
- Pbkdf2

- Totp

## Installation

```shell
npm install @gtsc/crypto
```
