# Variable: JwtAlgorithms

> `const` **JwtAlgorithms**: `object`

The cryptographic algorithms supported for JSON Web Tokens and JSON Web Keys.

## Type declaration

### HS256

> **HS256**: `string` = `"HS256"`

HMAC using SHA-256.

### EdDSA

> **EdDSA**: `string` = `"EdDSA"`

EdDSA using Ed25519.
